# WebSocket Audio Streaming Server

Сервер для потоковой передачи аудио в реальном времени через WebSocket.

## Возможности

- Потоковая передача аудио от мобильного устройства
- Прослушивание стримов через веб-панель
- Поддержка множественных слушателей
- Автоматическая очистка неактивных стримов

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на порту 3000 (или PORT из переменных окружения).

## API

### WebSocket команды

**Начать стрим:**
```json
{
  "type": "start_stream",
  "deviceId": "your-device-id"
}
```

**Слушать стрим:**
```json
{
  "type": "listen_stream",
  "deviceId": "device-id-to-listen"
}
```

**Остановить стрим:**
```json
{
  "type": "stop_stream"
}
```
